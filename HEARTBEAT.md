# HEARTBEAT.md

默认保持轻量，但要优先维护记忆系统的可用性。

## 轮询时优先检查

1. `NOW.md` 是否需要刷新
   - 当前重点是否变了
   - 待跟进 / 阻塞是否过期

2. 今天的 `memory/YYYY-MM-DD.md` 是否需要补记
   - 是否有重要操作还没落盘
   - 是否有容易丢失的上下文

3. 是否有内容值得提炼到 `knowledge/`
   - 新偏好
   - 新决策
   - 新流程
   - 新项目背景

4. 是否有未完成但容易遗忘的事项
   - 尤其是卡住太久、但仍重要的跟进项

## 原则

- 能轻量完成就轻量完成
- 不要凭空制造任务
- 不要把 daily 流水账整段搬进长期知识
- 如果没有值得处理的事，就保持安静

## 无事可做时

返回：`HEARTBEAT_OK`
